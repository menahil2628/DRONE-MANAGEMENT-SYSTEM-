# Drone Management System with Load Balancer

A Python-based Drone Management System that manages 
drone fleet operations and optimizes task distribution 
using load balancing and pathfinding algorithms.

##  Features
- Manages multiple drones simultaneously
- Load balancing for optimal task distribution
- Pathfinding algorithm for route optimization
- User-friendly GUI interface

##  Technologies Used
- Python
- Tkinter (GUI)
- Load Balancing Algorithm
- Pathfinding Algorithm

##  How to Run
1. Clone this repository
2. Run `python main.py`
3. Manage drone operations via GUI
